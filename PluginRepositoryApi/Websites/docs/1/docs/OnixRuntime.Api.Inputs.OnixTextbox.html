Class OnixTextbox

Namespace
OnixRuntime.Api.Inputs
Assembly
OnixRuntime.dll

Convenience class to have the basic functionality of a textbox without doing the hard work that comes with it. You're welcome. Set IsFocused to true to allow the user to start typing in it. don't forget that the user won't be able to use their keyboard until IsFocused is false! Use the Render function to render the textbox.
