GetPathList(string, bool)

Gets a list of paths in the pack. This will return an empty array if the path does not exist or is not a directory. Note that internally, this will ALWAYS recurse. The difference is that the output will be filtered. So if you plan on recursing, you should not go folder by folder and manually do it.
