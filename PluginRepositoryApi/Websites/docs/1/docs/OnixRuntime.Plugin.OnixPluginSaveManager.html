TrackModule(OnixModule)

Tracks the given module if it's not already tracked. This means next save or load, the module's save and load methods will be called. Calling this will also try to load the module from the last save.

TrackSetting(OnixSetting, bool, string)

Tracks the given setting if it's not already tracked. Tracked settings will have their values saved and restored with the module's configuration.

public void TrackSetting(OnixSetting setting, bool addParentModuleNameAsPrefix = true, string prefix = "")

setting OnixSetting

The setting to track.
